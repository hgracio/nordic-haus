# Images Directory

## Nordic Haus Logo

Please save the Nordic Haus logo image as:
- **Filename**: `nordic-haus-logo.png`
- **Location**: This directory (`/images/`)
- **Format**: PNG (recommended for transparency)
- **Recommended size**: 400x400 pixels or larger

The logo is used in:
1. Header navigation (top left)
2. Footer (center)
3. Chatbot avatar (bot messages)

The website includes fallback placeholders if the logo file is not found.
